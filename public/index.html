API Documentation

Complete reference for the WebNesti API

Base URL: https://api.webnesti.com

Authentication

POST /api/auth/register Register a new account
POST /api/auth/login Sign in and get JWT token
GET /api/auth/me Get current user profile

Chat Completions

POST /v1/chat/completions Create chat completion (OpenAI-compatible)

Models

GET /v1/models List all available models
GET /v1/models/:id Get model details

API Keys

GET /api/keys List all API keys
POST /api/keys Create a new API key
DELETE /api/keys/:id Revoke an API key

Billing & Usage

GET/api/usageGet usage statistics
GET/api/billing/transactionsList transactions
POST/api/billing/topupInitiate top-up

Pricing & Budget

GET/api/pricingGet all pricing info
GET/api/pricing/currentGet current plan
POST/api/pricing/upgradeUpgrade plan
GET/api/budgetGet budget settings
PUT/api/budgetUpdate budget
GET/api/budget/alertsGet budget alerts
GET/api/calculateCalculate cost

Referral

GET/api/referralGet referral info
POST/api/referral/applyApply referral code

System

GET/healthHealth check
GET/v1/openapi.jsonOpenAPI specification
